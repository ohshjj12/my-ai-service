How it works

사용 방법

4단계만 거치면 분석 결과를 받을 수 있습니다.

1

회원가입

이메일과 비밀번호로 계정을 생성합니다.

2

로그인

API 요청 시 Basic Auth 인증 정보를 포함합니다.

3

이미지 업로드

초음파 이미지를 multipart/form-data로 전송합니다.

4

결과 확인

예측 성별과 신뢰도를 JSON으로 즉시 받습니다.

API Reference

주요 API 엔드포인트

Base URL: https://my-ai.kpearl.net

POST /api/auth/signup 
# Request Body (JSON)
{
  "username": "hong",
  "password": "password123",
  "email": "hong@example.com"
}

# Response 201
{
  "message": "User registered successfully",
  "username": "hong"
}
POST /api/predictions/analyze 
# Request (multipart/form-data)
Authorization: Basic {base64}
file: <image file>

# Response 200
{
  "predictedGender": "MALE",
  "confidenceScore": 0.9231,
  "analyzedAt": "2026-03-26T..."
}
GET /api/predictions/my-results 
# Request Headers
Authorization: Basic {base64}

# Response 200
[
  {
    "predictionId": 1,
    "originalFileName": "scan.jpg",
    "predictedGender": "FEMALE",
    "confidenceScore": 0.8754
  }
]
GET /api/predictions/my-results 
# Basic Auth 생성 방법
username:password → Base64 인코딩

# curl 예시
curl -u hong:password123 \
  -F "file=@scan.jpg" \
  https://my-ai.kpearl.net
    /api/predictions/analyze
