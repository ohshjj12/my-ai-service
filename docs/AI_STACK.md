## 1. 텍스트 / 리포트 생성

| 항목 | 내용 |
|---|---|

| 항목 | 내용 |

## 2. 이미지 분석

### GPT-4o Vision (OpenAI)

| 항목 | 내용 |
|---|---|
| 용도 | 초음파 이미지 → 상태 설명 텍스트 생성 |
| 별도 설치 | 없음 (API 호출) |
| 비용 (512×512 이하, low detail) | $0.0002 / 장 (0.3원) |
| 비용 (1024×1024, high detail) | $0.0019 / 장 (2.8원) |
| 비용 (2048×2048, high detail) | $0.007 / 장 (10원) |
| 장점 | 모델 학습 없이 바로 사용 가능 |
| 단점 | 의료 이미지 특화 아님, 정확도 한계 |

### YOLOv8 (Ultralytics) — 자체 학습 모델

| 항목 | 내용 |
|---|---|
| 용도 | 초음파 이미지에서 신체 부위 객체 감지 |
| 설치 | `pip install ultralytics` |
| API 비용 | **없음** (자체 서버 운영) |
| 서버 비용 | GPU 서버 필요 (AWS g4dn.xlarge: 약 $0.5/시간) |
| 학습 데이터 | 직접 수집 또는 공개 데이터셋 필요 |
| 장점 | 속도 빠름, 커스터마이징 가능 |
| 단점 | 학습 데이터 수집/라벨링 필요 |

### EfficientNet / ResNet (PyTorch) — 자체 학습 모델

| 항목 | 내용 |
|---|---|
| 용도 | 이미지 분류 (상태 정상/비정상, 주차 추정 등) |
| API 비용 | **없음** (자체 서버 운영) |
| 장점 | 적은 데이터로 fine-tuning 가능 |
| 단점 | 학습 파이프라인 구축 필요 |

## 3. 영상 분석

### GPT-4o Vision + OpenCV 프레임 추출

| 항목 | 내용 |
|---|---|
| 용도 | 영상 → 프레임 추출 → GPT Vision으로 분석 |
| 비용 (1프레임/10초, 1분 영상) | 약 $0.011 / 건 (16원) |
| 비용 (1프레임/5초, 1분 영상) | 약 $0.023 / 건 (33원) |
| 비용 (1프레임/초, 1분 영상) | 약 $0.11 / 건 (160원) |
| 추천 샘플링 | **10초당 1프레임** (비용/품질 균형) |
| 장점 | 모델 학습 불필요, 빠른 MVP |
| 단점 | 영상 파일 직접 지원 안 함, 프레임 추출 코드 필요 |

### MediaPipe (Google)

| 항목 | 내용 |
|---|---|
| 용도 | 영상에서 자세/골격 추정, 움직임 패턴 추출 |
| API 비용 | **없음** (오픈소스, 로컬 실행) |
| 설치 | `pip install mediapipe` |
| 장점 | 실시간 처리 가능, 경량 |
| 단점 | 일반 인체 기준 → 태아 적용 시 커스텀 필요 |

### OpenCV + YOLOv8 조합

| 항목 | 내용 |
|---|---|
| 용도 | 프레임별 객체 감지 → 움직임 변화량 계산 |
| API 비용 | **없음** (자체 서버 운영) |
| 장점 | 가장 현실적인 영상 분석 접근법 |
| 단점 | GPU 서버 필요, 모델 학습 선행 |

## 5. MVP 단계별 추천 조합

### STEP 1 — 지금 당장 (모델 학습 없이)

```
이미지 분석: GPT-4o Vision API
텍스트 리포트: GPT-4o / GPT-4o mini
영상 분석: OpenCV 프레임 추출 → GPT-4o Vision
```

**월 비용 추정 (사용자 100명 기준)**

| 항목 | 건수 | 단가 | 월 비용 |
|---|---|---|---|
| 이미지 분석 | 300건 | 3원 | 900원 |
| 텍스트 리포트 | 300건 | 6원 | 1,800원 |
| 영상 분석 (10초/1프레임) | 100건 | 16원 | 1,600원 |
| **합계** | | | **≈ 4,300원** |

---

### STEP 2 — 자체 모델 도입 후

```
이미지 분석: YOLOv8 (커스텀 학습) — GPU 서버 자체 운영
텍스트 리포트: GPT-4o mini (비용 절감)
영상 분석: OpenCV + YOLOv8
```

**서버 비용 추정**

| 항목 | 사양 | 월 비용 |
|---|---|---|
| AI 추론 서버 | AWS g4dn.xlarge (GPU) | $380 (~55만원) |
| 일반 API 서버 | AWS t3.medium | $34 (~5만원) |
| GPT-4o mini (리포트) | 1,000건 | $0.5 (~730원) |

> 사용자 수 늘수록 자체 모델이 훨씬 저렴
